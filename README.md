# 📊 Retail Performance Dashboard

## 🔹 Introduction

This project is an interactive Excel dashboard created to analyze retail sales data using pivot tables and slicers.

## 🔹 Problem Understanding

Businesses find it difficult to analyze large sales data and identify trends, categories, and customer behavior.

## 🔹 Approach

* Data cleaning and preparation
* Created calculated column (Total = Price × Quantity)
* Used Pivot Tables for summarization
* Built charts for visualization
* Added slicers for interactive filtering
* Designed a user-friendly dashboard

## 🔹 Dashboard Overview

The dashboard includes:

* Total Revenue
* Total Orders
* Average Order Value
* Max Sale
* Revenue by Category
* Revenue by Mall
* Revenue by Gender

## 🔹 Key Insights

* Clothing is the top-performing category
* Revenue shows growth over time
* Certain malls generate higher revenue
* Customer purchasing behavior varies

## 🔹 Conclusion

This dashboard helps in quick decision-making and provides clear insights into business performance.
